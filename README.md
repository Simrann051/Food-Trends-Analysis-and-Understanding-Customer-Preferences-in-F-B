# 🍽️ Food Trends Analysis - Power BI Project

## 📌 Project Overview
This project analyzes restaurant data to understand customer preferences, food trends, and business insights in the Food & Beverage industry.

## 📊 Tools Used
- Power BI
- DAX
- Power Query
- Dataset: Zomato (Kaggle)

## 🔍 Key Analysis
- Food Trends Analysis
- Sentiment Analysis
- Customer Segmentation
- Product Insights
- Strategy Dashboard

## 📈 Key Insights
- North Indian cuisine dominates customer engagement
- Mid-range pricing generates highest sales
- Customer satisfaction is mostly moderate
- Silent users form majority of customers

## 🧠 Features
- Interactive dashboards
- KPI metrics
- Sentiment classification
- Business recommendations
